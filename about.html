About The Project This project was originally created for a website design course at Ashland University. It is meant to be a very simple tool that allows for quick unit conversion at your fingertips, both on desktop and mobile.
How was it built? This project employs Material design elements with a responsive layout. This is made available through javascript and css libraries provided by the Materialize Project.
Licensing Information The Materialize project is released under the MIT License. In accordance with the academic nature of this project, I have decided to release all of the code under that license as well.
